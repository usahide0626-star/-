🎯 主要プロジェクト

1. 打者成績予測AI

機械学習を使用した次シーズンの打者成績予測システム。過去5年のデータを分析し、90%の精度で予測を実現しました。

  • 技術: Python, Pandas, Scikit-learn, Django
  • 成果: 予測精度90%以上を達成

2. リアルタイム野球データダッシュボード

最新の試合結果とプレイヤー統計をリアルタイムで表示するWebダッシュボード。

  • 技術: React, D3.js, REST API
  • 成果: 月間1万PVを達成

3. 野球スタッツ分析ツール

複数リーグの選手成績を一括分析・比較できるツール。CSVインポート機能付き。

  • 技術: Vue.js, Flask, SQLite
  • 成果: 野球コミュニティで人気ツールに
